¿Qué es GitHub?

GitHub es un servicio de alojamiento de proyectos que utiliza el sistema de control de versiones Git. Permite a los desarrolladores colaborar, compartir y contribuir a proyectos de código abierto y privado.

Características

  • Control de versiones con Git
  • Colaboración en tiempo real
  • Integración continua y despliegue
  • Documentación y gestión de proyectos
